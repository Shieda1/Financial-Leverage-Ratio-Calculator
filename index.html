What is the financial leverage ratio calculator?
The financial leverage ratio calculator is a tool that determines how much of the company's assets are financed using debt instead of equity.
How to calculate financial leverage ratio?
To calculate financial leverage ratio you should use the following formula:
financial leverage ratio = (current assets + non-current assets) / total equity
Why use financial leverage ratio calculator?
The financial leverage ratio calculator is used in order to compute how much of the company's assets are financed using debt instead of equity.
